What We Built

A fully functional client portal connected live to ShopWorks, built and ready for deployment.

📦

Live Order Dashboard

Clients see their orders pulled directly from ShopWorks in real time. Organized into tabs: All, Active, Shipped, and On Hold.

🔐

Secure Login System

Each client has a unique email and password. Sessions are secured with encrypted tokens that expire after 8 hours.

🎛️

Admin Management Panel

PXP staff can add, edit, and remove client accounts from a dedicated admin interface — no database access needed.

🔭

Scoped Order Visibility

Each client only sees their own company's orders. Admins can further restrict access to specific contacts within a company.

How It Works

End-to-end flow from onboarding a client to them viewing orders.

1

PXP staff adds a client in the Admin Panel

Enter the client's email, company name (matched to ShopWorks), and optionally set a password directly or send an invite link.

2

Client receives access

Either via an invite link (they set their own password) or with credentials set by admin. No IT involvement required.

3

Client logs in and sees their orders

The portal pulls orders from ShopWorks filtered to their company. The dashboard shows live production status, ship dates, tracking numbers, and more.

4

Client filters, exports, and prints

Clients can filter by date, status, contact, or design. They can export to CSV or print directly from the browser.

User Types

Two distinct access levels in the current system.

Client User

  • Logs in with email and password
  • Sees their own company's orders only
  • Can filter, export, and print orders
  • Cannot see other clients or admin panel

PXP Admin

  • Full client management (add/edit/delete)
  • Set passwords or send invite links
  • Control which orders each client can see
  • Access to both dashboard and admin panel

Proposed Features

Ideas to discuss — organized by impact and effort.

High Priority

Email Notifications on Status Change

Automatically notify clients when their order ships, goes on hold, or hits a key milestone. Reduces inbound status calls to PXP staff.

High Priority

Order Detail Page

Clicking an order opens a full detail view with complete history, all status flags, tracking links, and any notes — without squinting at the table.

Medium

Cross-Company Order View

A special access mode for clients managing multiple brands or locations — lets them see all their companies' orders in one view. (Already in development.)

Medium

Analytics Dashboard

Charts showing order volume over time, on-time shipment rate, orders by type, and status breakdowns — so clients can spot patterns at a glance.

Medium

Customer Notes / Messaging

A simple thread on each order where PXP staff can leave notes that clients can see. Replaces back-and-forth email for common questions.

Medium

Multi-Year Order History

Currently shows the current year only. Extend to show past years with a date range picker so clients can look up older orders without contacting PXP.

Lower Priority

Saved Filter Presets

Let clients save commonly used filter combinations (e.g., "my Dallas orders, not shipped") and jump back to them with one click.

Lower Priority

Branded Client Portals

Ability to show a client's logo alongside PXP branding on their portal — makes the tool feel like a white-label service offering for larger accounts.

Deployment
